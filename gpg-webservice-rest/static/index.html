Quick Start

POST /openai/register_user

Register a new user with automatic GPG key generation

POST /openai/sign_text

Sign text content using user's private GPG key

POST /openai/verify_text_signature

Verify a signature against a public key

POST /openai/encrypt_text

Encrypt text for a recipient using their public key

POST /openai/decrypt_text

Decrypt text using user's private key

GET /openai/function_definitions

Get all available functions for AI agent integration

Authentication

Most endpoints require authentication via API key. Include your API key in the X-API-KEY header. API keys are returned once upon user registration - store them securely.

üîí Security Notice: Never share your API keys or commit them to version control. All cryptographic operations use isolated temporary keyrings for maximum security.
